Installation

Holger Pandel 18 Juni 2018 avnavTouch
Vorbedingungen

Für die allererste Inbetriebnahme wird eine Tastatur benötigt, da direkt im Anschluss an den ersten Start ein paar grundlegende Konfigurationseinstellungen durchgeführt werden müssen. Dafür wird ist eine Tastatur nötig.

Speicherkarte

Es wird eine SD Karte mit mindestens 8 GB Speicher benötigt. Aber je größer, desto besser, aus zwei Gründen:

a) es sollte genug Platz für die verwendeten Seekarten vorhanden sein
b) je größer die Karte, desto schneller in der Regel ist sie auch

Ablauf
  1. SD Karte Formatieren, am besten mit SD Formatter 4.0
  2. heruntergeladene ZIP Datei direkt auf die Karte entpacken
  3. Karte über Windows ordentlich aushängen, in den Raspi stecken, booten
  4. im erscheinenden Menü unten die Sprache auf "Deutsch" ändern
  5. ein Häkchen vor "avnavTouch" setzen
  6. auf Installieren klicken, warten bis fertig, booten, freuen
Empfehlung

Es ist evtl. günstig, noch ein weiteres System auf der Karte zu installieren, um im Fehlerfall noch booten zu können. Sofern am Installationsort ein WLAN-Netz bereit steht, lässt sich das ganz leicht bereits während der Grundinstallation erledigen:

  • im PINN Startdialog das WiFi-Kennwort des bestehenden WLANs eingeben
  • PINN lädt die Liste der online verfügbaren OS Installationen herunter
  • zusätzlich zu avnavTouch ein OS auswählen, ich empfehle DietPi (schön klein und kann alles, was nötig ist)
  • Installation starten

PINN kümmert sich um alles weitere und am Ende hat man ein schönes Bootmenü und kann zwischen avnavTouch und dem Zusatz-OS wählen.
