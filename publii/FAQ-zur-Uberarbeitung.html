FAQ - zur Überarbeitung

Holger Pandel 18 Juni 2018
Allgemeine Info:

Die Basis für die Touch Oberfläche bilden der Browser Chromium im Kiosk Modus und die On-Screen Tastatur OnBoard. Das X Windows System wird hier nach autologin des Benutzers pi direkt gestartet, verwendet also keinen eigenen Display Manager, was es einfach macht.

Kann ich avnavTouch auch auf einem älteren Raspberry Pi 3 (ohne B+) installieren?

Ja, das geht. Ich habe die Installation soweit getestet, dass sie läuft. Allerdings war der alte Raspi firmwareseitig upgedated. Man kann im PINN Installer allerdings auf eine ältere Firmware beim Booten umstellen, sofern der Raspi zicken sollte. Ob dann aber noch alle Komponenten, wie WLAN, etc. gehen, kann ich nicht sagen. Einfach mal ausprobieren.

Wo finde ich die Anpassungen zum originalen Avnav?
  • das Basisbetriebssystem ist via apt Upgrade Mechanismen von mir direkt ohne Neuinstallation auf Raspbian Stretch hochgezogen worden
  • über offizielle Quellen installiert Pakete lassen sich in der /var/log/apt/history.log nachvollziehen, da habe ich keine extra Liste gepflegt
  • alle weiteren Änderungen finden sich im Ordner /home/pi/source:
    hier befindet sich eine an den Root-Ordner angelehnte Struktur, in der man sehen kann, welche Dateien ich wo neu angelegt, geändert oder sonstwie habe. BITTE NICHT LÖSCHEN! Dieser Ordner wird auch dazu verwendet, die Konfigurationsdateien für Headless- und Touch-Mode bereit zu stellen!
  • es gibt dort einen Ordner "src", hier befinden sich sämtliche als Quellen heruntergeladene Softwarepakete, die ich zusätzlich draufgepackt habe, wie onboard, rtl-ais, etc., da ich direkt auf dem Raspi komiliert habe.
Wie komme ich in die Konfigurationstool auf dem zweiten Desktop?

Tastatur anschließen, STRG-ALT-PfeilNachRechts drücken

Wie komme ich an das Konfigurationstool, das beim ersten Start geöffnet wurde?

Dazu gibt es mehrere Möglichkeiten:

  • im Touchbetrieb auf den zweiten Desktop wechseln, dort läuft es bereits
  • per SSH einloggen und auf der Konsole folgendes starten:

    sudo avnav_setup.sh
Wie schalte ich zwischen Touch Modus und reinem Serverbetrieb (headless) um?

Einfach auf dem zweiten Desktop in das Konfigurationstool wechseln und unter dem entsprechenden Menüpunkt umstellen. Beim nächsten Neustart steht der gewählte Modus zur Verfügung.

Wie kann ich avnav upgraden, wenn Andreas eine neue Version bereitstellt?

Einfach im Konfigurationstool den entsprechenden Menüpunkt starten, dann die zu installierende Version auswählen und das Update durchführen.

ACHTUNG: es erfolgt keine weitere Prüfung. Wenn euch das Update irgendwas eigengefummeltes zerschießt, müsst ihr selber suchen.

Wie mache ein Update des Betriebssystems?

Geht auch via Konfigurationstool. Einfach den entsprechenden Menüpunkt auswählen. Oder aber via Kommandozeile per SSH:

sudo apt update
sudo apt list --upgradable
sudo apt upgrade
sudo apt dist-upgrade
